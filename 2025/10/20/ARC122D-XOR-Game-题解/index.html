ARC122D XOR Game 题解

link 本题为原T1

黑板上有 2n2n2n 个整数。
每轮：

  1. Alice 选一个数 xxx 擦掉；

  2. Bob 选一个数 yyy 擦掉；

  3. 将 x⊕yx \oplus yx⊕y 记在笔记本上。（是另一个地方！）

重复 nnn 轮，笔记本上有 nnn 个数，分数是它们的最大值。
Alice 想最大化分数，Bob 想最小化分数。双方最优策略下，求分数。

1≤n≤2×105,0≤Ai≤2301\leq n\leq 2\times10^5,0\leq A_i\leq 2^{30}1≤n≤2×105,0≤Ai​≤230

这道题其实不是博弈论。

因为最后所有数都能一一配对。当 Alice 选择某一组的其中一个数时，Bob 一定可以选择对应的另外一个数。

所以 Alice 怎么选根本不重要，Bob 占据绝对主动，一定可以最小化答案。

（赛时居然只看出来了第一步 A 怎么选不重要，太菜了qaq）

那么这道题的本质是：给你 2n2n2n 个数，问你如何两两配对，使每组数的异或的最大值最小。

异或这种东西吧，还是要按位考虑的。不要成天想着哪些位置反掉了，然后根据修改的更新。

我们从高位到低位考虑贪心，尽量使高位为零。

手玩可以发现，如果最高位所有的 111 都可以在一起消掉，那么就可以使这些 111 以特定的方式相消使得这一位结果为 000。

也就是说，如果 111 的数量为偶数，就可以让下一位变为“最高位”了，即递归下去。

当然这里递归下去是要分为这一位是 000 和 111 两个集合递归。毕竟 111 是只能与 111 消的。

−\color{transparent}\tiny -−

那么如果 111 的数量为奇数，就必然有一个 111 无法找 111 相消，它就使得结果在这一位上为 111。

由于其他 111 都能消掉，在这一位上为 000，那么说，实际上其他 111 是对结果没有影响的。直接以上面那玩意为结果就可以了。

考虑如何求出 000 的集合与 111 的集合相消的最小值。使用字典树即可。

那么这一位由于已经是最大值，就不用递归下去了。

最后答案取 max⁡\maxmax 即可。
